If you are seeing this page, you have done a GET request instead of a POST request.

Please do a POST request in order for your login form to work

For your form to work, the email input must have the name attribute set to email, and the password input must have the name attribute set to password. For example:

<input type="email" name="email"/> and <input type="password" name="password"/>
